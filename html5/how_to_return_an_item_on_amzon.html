How to Return an Item on Amzon

Learn how to return an item purchased on Amazon using the website or mobile app.
  1. Go to the Amazon website (www.amazon.com) or open the Amazon mobile app.
    Log in to your Amazon account if you are not already signed in.
  2. Navigate to Your Orders.

    On the website, click Returns & Orders at the top right. In the app, tap the menu icon and select Your Orders.

  3. Locate the item you want to return and click Return or Replace Items.

    If the option is unavailable, check the return eligibility period for the item.

  4. Select the reason for the return from the dropdown menu.

    Choose the reason that best describes your situation, such as "Defective item" or "Wrong item sent."

  5. Choose your preferred return option.

    You can opt for a refund, replacement, or repair, depending on the item and return policy.

  6. Choose how you will return the item.

    Select a return method, such as dropping it off at an authorized location or scheduling a pickup.

  7. Print the return label and packing slip if required.
    Follow the instructions provided by Amazon for packaging and labeling the item.
  8. Package the item securely and attach the return label.
    Ensure the item is packed according to Amazon's return instructions to avoid damage during transit.
  9. Drop off the package at the selected location or have it picked up as scheduled.
    Retain the return receipt or tracking number for reference.
  10. Monitor your return status in Your Orders.
    Amazon will update you on the status of your return and refund or replacement.
